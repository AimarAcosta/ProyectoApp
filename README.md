# ProyectoApp

## Descripción
Proyecto desarrollado para el ciclo de Desarrollo de Aplicaciones Multiplataforma.
Es una herramienta diseñada para sustituir los registros manuales de inventario en una pequeña empresa (ej. hostelería). La aplicación operará en un dispositivo tipo Tablet anclado a la pared, permanentemente encendido y conectado a la corriente.

## Requisitos Técnicos
* **Lenguaje:** Kotlin
* **Base de Datos Local:** Room 
* **Arquitectura:** MVC estricto
* **Control de Versiones:** GitHub
